# База данных "Университет" - Расширенная версия

## Описание проекта
Проект базы данных для управления учебным процессом с автоматическим развертыванием через Docker.

## Особенности реализации:
-  4 таблицы со связями
-  3 нормальная форма
-  Автоматическое заполнение случайными данными (50 студентов, 200 оценок)
-  Docker Compose для развертывания
-  Триггеры для валидации данных
-  Индексы для оптимизации производительности
-  Тесты производительности (EXPLAIN ANALYZE)
-  Constraints (NOT NULL, UNIQUE, CHECK)
-  Примеры сложных запросов (GROUP BY, WHERE, HAVING)

## Файлы проекта:
- `database.sql` - полный код БД с триггерами и индексами
- `ERD.md` - диаграмма связей и описание архитектуры
- `docker-compose.yml` - конфигурация для развертывания
- `README.md` - эта инструкция
